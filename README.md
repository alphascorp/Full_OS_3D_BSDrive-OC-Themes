# Full_OS_3D_BSDrive-OC-Themes

---

<h4 align="center">Themes for use with OpenCore 0.7.0 and Higher and OpenCore Legacy Patcher (OCLP)</h4>

---

## :warning: First Use ➤  Change `config.plist` in `EFI/OC`

### Set PickerVariant (Full_OS_3D_BSDrive_SequoiaLight theme for example)

- Misc —> Boot —> PickerVariant: Alphascorp\Full_OS_3D_BSDrive_SequoiaLight

### Set PickerAttributes

- Misc —> Boot —> PickerAttributes: 145

### Like this: :arrow_heading_down:
```
<key>Misc</key>
<dict>
  
  …
  
  <key>Boot</key>
  <dict>
    
    …
    
    <key>PickerAttributes</key>
    <integer>145</integer>
    
    …
    
    <key>PickerVariant</key>
    <string>Alphascorp\Full_OS_3D_BSDrive_SequoiaLight</string>
    
    …
          
  </dict>
```

## Replace the unziped `Resources` folder in `EFI/OC`
